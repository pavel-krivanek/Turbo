# Turbo
Text based interface for Pharo

## How to load 

```smalltalk
Metacello new
  baseline: 'Turbo';
  repository: 'github://pavel-krivanek/Turbo';
  load.
```

or with VTE (+Gtk)

```smalltalk
Metacello new
  baseline: 'TurboWithVte';
  repository: 'github://pavel-krivanek/Turbo';
  load.
```
